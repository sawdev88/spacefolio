Awesome Easy CSS Spinners

Here's a quick write up for a spinner that's so awesome and lightweight 'everybody' will be talking about it. First off, this thing is CSS only, no JS needed (bummer). It does use CSS animations though, so >== ie10. Lets break this thing down. * I use rem (no px) for sizing. Scales great.

We'll start off with the HTML. We'll just use an empty div for span's sake.


        <div></div>

Now we'll create a generic and reusable class called 'spinner'. This class is going to give the div a fixed width/height and then a border radius to make our div into a perfect circle. 


        <div class="spinner"></div>

        <style>
          .spinner {
            border: .5rem solid #ddd;
            height: 6rem;
            width: 6rem;
            border-radius: 50%;
          }
        </style>

So far so good. We have a pretty sweet circle but lets give it some color. We'll build off our current .spinner style. All we have to do is add a border-color to one of the sides. I prefer the top for no reason. Let's add border-top-color. 


        <style>
          .spinner {
            border: .5rem solid #ddd;
            height: 6rem;
            width: 6rem;
            border-radius: 50%;
            border-top-color: #008080;

            -webkit-animation: spin 1.5s linear infinite; /* Safari 4+ */
            -moz-animation:    spin 1.5s linear infinite; /* Fx 5+ */
            -o-animation:      spin 1.5s linear infinite; /* Opera 12+ */
            animation:         spin 1.5s linear infinite;
          }
        </style>

Let's create a keyframe. 


        <style>
          @keyframes spin {
            from { transform: rotate(0deg); }
            to { transform: rotate(360deg); }
          }
        </style>

Awesome. Now let's get this thing spinning. We're going to add an animation prop (and it's prefixes) to our spinner class. 


        <style>
          .spinner {
            border: .5rem solid #ddd;
            height: 6rem;
            width: 6rem;
            border-radius: 50%;
            border-top-color: #008080;

            -webkit-animation: spin 1.5s linear infinite; /* Safari 4+ */
            -moz-animation:    spin 1.5s linear infinite; /* Fx 5+ */
            -o-animation:      spin 1.5s linear infinite; /* Opera 12+ */
            animation:         spin 1.5s linear infinite;
          }

          @keyframes spin {
            from { transform: rotate(0deg); }
            to { transform: rotate(360deg); }
          }
        </style>

There it is... An awesome spinner. Not a fan of that linear spin? Just remove 'linear' from the animation prop and get a cool little slow/fast spin. Want to spin counter clockwise? Just change the keyframe from '360deg' to '-360deg'. 


        <style>
          .spinner {
            border: .5rem solid #ddd;
            height: 6rem;
            width: 6rem;
            border-radius: 50%;
            border-top-color: #008080;

            -webkit-animation: spin 1.5s infinite; /* Safari 4+ */
            -moz-animation:    spin 1.5s infinite; /* Fx 5+ */
            -o-animation:      spin 1.5s infinite; /* Opera 12+ */
            animation:         spin 1.5s infinite;
          }
        </style>
